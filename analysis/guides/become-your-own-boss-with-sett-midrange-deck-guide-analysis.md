## Summary
Sett midrange value deck guide. Body-Order legend using buffs to keep powerful units alive longer. The Boss legend ability preserves units. Key battlefield: Monastery of Hirana (draw on conquer). Champions: Sett-Brawler, Qiyana-Victorious, Fiora-Victorious.

## Key Facts
Sett = Body + Order domains; midrange/value strategy; The Boss = legend ability keeps units alive; Monastery of Hirana = battlefield draws a card on conquer (huge); Pit Rookie + Kinkou Monk buff better units; Arena Bar gear = feign death; Cannon Barrage = AoE wipe; Call to Glory = surprise buff opponents must play around; Sett-Brawler + Showstopper = 7-8 might threat; based on Vendorty's 1st place RIftlab Trials list
