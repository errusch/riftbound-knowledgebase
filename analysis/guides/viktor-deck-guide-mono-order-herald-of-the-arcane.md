## Summary
Viktor (Herald of the Arcane) Mono-Order deck guide. Order-Mind legend running zero Mind runes. Aggressive token-based deck using Recruit tokens, on-play/Deathknell effects, Hidden Blade draw engine. Strong early-game tempo.

## Key Facts
Viktor = Order + Mind domains; Mono-Order build (0 Mind runes); aggressive unit + removal deck; creates Recruit tokens; Hidden Blade = draw engine + removal; Cull the Weak = removal; Cruel Patron = above-rate unit; Watchful Sentry + Ravenbloom Student = only Mind cards; Consult the Past = +1 draw; Grand Strategem + Siphon Power = endgame hold tools; Harnessed Dragon = expensive but kills best opposing unit; battlefields: Obelisk of Power, Reaver's Row, Trifarian War Camp; won first TTS tournament
