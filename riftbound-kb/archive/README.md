# Archive Policy

Purpose
This directory is the canonical destination for stale, superseded, or supporting Riftbound material that should not live in the main trust path.

Current state
- Legacy Riftbound material outside `riftbound-kb/` has been inventoried and mapped but not yet bulk-moved here.
- Until that move happens, agents should treat `riftbound-kb/` as the canonical source and older Playground paths as supporting/legacy only.

Rule
- Do not delete legacy material without approval.
- When legacy material is moved here, preserve provenance and original paths.
