# Riftbound KB

Purpose: canonical private source of truth for Riftbound so any agent can operate like a Riftbound expert.

Priority order:
1. factual accuracy
2. current official tournament meta
3. retrieval / structure
4. future content reuse

This subtree is the canonical destination for reviewed Riftbound knowledge.
Legacy material elsewhere in Playground may still be useful, but it is not automatically canonical.

Top-level layout
- canon/cards/ — canonical card records
- canon/rules/ — official rules and tournament policy
- canon/rulings/ — errata and clarifications
- events/ — official event records and results
- decklists/ — event-linked decklists and best-of-legend lists
- meta/ — dated tournament-meta snapshots
- market/ — trusted market references and trends
- reference/ — neutral primers, glossary, evergreen retrieval docs
- sources/ — source registry, trust policy, coverage indexes
- archive/ — retired or superseded Riftbound material moved out of the main trust path
- _work/ — temporary planning, inventory, mapping, ingest artifacts

Decision rules
- Official beats secondary
- Secondary beats inferred
- Every non-trivial claim should be source-labeled
- Unsupported claims should not appear as canonical truth
- Old/stale/conflicting files get archived, not deleted without approval
